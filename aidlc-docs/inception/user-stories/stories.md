# User Stories - 保活手帳 v1 SLC

## Story Map Overview

ユーザージャーニーに沿ったエピック分類:

```
見学前          見学中            見学直後          申込時
(リストアップ)   (記録する)        (整理する)        (思い出す)
   |               |                |                |
   v               v                v                v
 園を追加      メモを書く       メモを編集       一覧で見返す
 見学日設定    見学のコツ表示   園情報を修正     メモを読む
 アプリ説明    (初回メモ時)     見学日を更新
                                園を削除
```

---

## Epic 1: 見学前 - リストアップ

### US-01: 園を追加する
**As a** 保活中の保護者
**I want to** 通える範囲の園を名前で追加したい
**So that** 見学候補のリストを作れる

**Acceptance Criteria:**
- Given 園一覧画面を表示している
  When 「園を追加する」ボタンをタップする
  Then 園追加画面に遷移する
- Given 園追加画面で園名を入力している
  When 「追加する」ボタンをタップする
  Then 園が追加され、園一覧画面に遷移し、追加した園が一覧の先頭に表示される
- Given 園追加画面で園名が空の状態
  When 画面を確認する
  Then 「追加する」ボタンが非活性である

### US-02: 見学日を設定する
**As a** 保活中の保護者
**I want to** 園を追加するときに見学日を設定したい（任意）
**So that** いつ見学するか/したかを記録できる

**Acceptance Criteria:**
- Given 園追加画面を表示している
  When 見学日フィールドを確認する
  Then デフォルトは「未定」が選択されている
- Given 園追加画面で見学日を選択している
  When 日付を選んで追加する
  Then 選択した日付が見学日として保存される

### US-03: アプリの使い方を知る
**As a** 保活中の保護者
**I want to** 初めてアプリを開いたときに何ができるか知りたい
**So that** 迷わず使い始められる

**Acceptance Criteria:**
- Given 初めてアプリを起動した
  When アプリが表示される
  Then アプリの説明（できること・データの保存先）がオンボーディング画面として表示される
- Given オンボーディング画面が表示されている
  When 「はじめる」をタップする
  Then 園一覧画面に遷移し、次回起動時にはオンボーディングは表示されない

## Epic 2: 見学中 - 記録する

### US-04: メモを書く
**As a** 保活中の保護者
**I want to** 見学中に気づいたことをサッとメモしたい
**So that** 印象が鮮明なうちに記録を残せる

**Acceptance Criteria:**
- Given 園一覧画面を表示している
  When 園カードをタップする
  Then 園詳細画面に遷移し、2タップ以内でメモ入力を開始できる
- Given 園詳細画面でメモが空の状態
  When メモ欄を確認する
  Then プレースホルダー「気づいたことを自由に書けます」が表示される

### US-05: 見学のコツを知る
**As a** 保活中の保護者
**I want to** 初めて園の詳細を見るときに見学のコツを知りたい
**So that** 何に注目すればいいかのヒントが得られる

**Acceptance Criteria:**
- Given 初めて園詳細画面に遷移した（アプリ全体で初回のみ）
  When 園詳細画面が表示される
  Then 見学のコツ（3つ）がヒントとして表示される
- Given 見学のコツが表示されている
  When 閉じる操作をする
  Then ヒントが閉じ、園詳細画面が通常表示される
- Given 一度見学のコツを閉じた後
  When 同じ園や別の園の詳細画面に遷移する
  Then 見学のコツは自動表示されない（初回のみ）

## Epic 3: 見学直後 - 整理する

### US-06: メモを編集する
**As a** 保活中の保護者
**I want to** 見学後に落ち着いてメモを追記・修正したい
**So that** 記録をより正確に残せる

**Acceptance Criteria:**
- Given 園詳細画面でメモが表示されている
  When [編集]をタップする
  Then メモがインライン編集モードになり、複数行のテキストを入力できる
- Given メモを編集している
  When 編集を完了する
  Then メモが保存され、更新日時が更新される

### US-07: 園名を編集する
**As a** 保活中の保護者
**I want to** 園名の誤字を修正したい
**So that** 正確な園名で管理できる

**Acceptance Criteria:**
- Given 園詳細画面で園名が表示されている
  When [編集]をタップして園名を変更する
  Then 園名が更新され、一覧にも反映される
- Given 園名を編集している
  When 園名を空にしようとする
  Then 空にはできず、バリデーションエラーが表示される

### US-08: 見学日を変更する
**As a** 保活中の保護者
**I want to** 見学日が変更になったときに更新したい
**So that** 正確な見学日を記録できる

**Acceptance Criteria:**
- Given 園詳細画面で見学日が表示されている
  When [編集]をタップする
  Then 日付ピッカーまたは「未定」を選択できる
- Given 見学日を変更した
  When 変更を確定する
  Then 見学日が更新され、一覧にも反映される

### US-09: 園を削除する
**As a** 保活中の保護者
**I want to** 誤って追加した園を削除したい
**So that** リストを正確に保てる

**Acceptance Criteria:**
- Given 園詳細画面を表示している
  When 「この園を削除」をタップする
  Then 確認ダイアログ「この園を削除しますか？」が表示される
- Given 確認ダイアログが表示されている
  When 「削除」をタップする
  Then 園が削除され、園一覧画面に遷移する
- Given 確認ダイアログが表示されている
  When 「キャンセル」をタップする
  Then ダイアログが閉じ、園はそのまま残る

## Epic 4: 申込時 - 思い出す

### US-10: 園一覧を見返す
**As a** 保活中の保護者
**I want to** 登録した園の一覧をざっと見返したい
**So that** どの園を見学したか全体像を把握できる

**Acceptance Criteria:**
- Given 複数の園が登録されている
  When 園一覧画面を表示する
  Then 園がカード形式で表示され、園名・見学日・メモの冒頭プレビューが確認できる
- Given 園が0件の状態
  When 園一覧画面を表示する
  Then 空状態メッセージが表示される

### US-11: メモを読んで思い出す
**As a** 保活中の保護者
**I want to** 特定の園のメモを読み返したい
**So that** 見学時の印象を思い出して判断材料にできる

**Acceptance Criteria:**
- Given 園一覧画面でメモのある園カードが表示されている
  When カードを確認する
  Then メモの冒頭がプレビュー表示されている
- Given 園カードをタップして園詳細画面を開いた
  When メモ欄を確認する
  Then メモの全文が表示される

## Epic 5: 共通 - サポート

### US-12: 見学のコツを再確認する
**As a** 保活中の保護者
**I want to** 見学のコツをもう一度見たい
**So that** 次の見学に備えられる

**Acceptance Criteria:**
- Given 園詳細画面を表示している
  When メモ欄の近くにある「見学のコツ」リンクをタップする
  Then 見学のコツ（3つ）が表示される

### US-13: アプリの説明を再確認する
**As a** 保活中の保護者
**I want to** アプリの使い方をもう一度確認したい
**So that** 操作に迷ったときに確認できる

**Acceptance Criteria:**
- Given 園一覧画面を表示している
  When ヘルプアイコン[?]をタップする
  Then アプリの説明（オンボーディングと同じ内容）が再表示される
